# SmartContract-Hyperledger-Golang

# Setup

## Setup hyperledger
In folders `./network/setup`. Run script follow below step

1. Install Pre-Requisites
- `./install-prereqs.sh`
- `exit`
- `vagrant ssh`

2. Install Fabric
- `sudo -E ./install-fabric.sh`
3. Install CA Server

## Setup tools

1. Install GoTools
- `./install-gotools.sh`
2. Install Hyperledger Explorer
- `./install-explorer.sh`
3. Install Node JS + Dev Utilities
